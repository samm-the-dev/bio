# Personal Website Project

## Overview
A personal website built with React and deployed to GitHub Pages. This site showcases projects, social profiles, and personal interests.

## Guiding Principles
- **Hand-authored content**: All body text/non-structural content is personally written, not AI-generated
- **AI-assisted design/build**: The design and implementation may be AI-assisted, but content remains authentic
- **Personal presentation**: A true reflection of interests, work, and personality
