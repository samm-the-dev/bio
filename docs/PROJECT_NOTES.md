## Scope Management

### V1 Priorities
**Must Ship:**
- Home with hero and navigation
- Projects page with code and games sections
- About page with bio
- Basic responsive design

**Should Include:**
- Now page
- At least one stats integration (Letterboxd or GitHub)

**Nice to Have:**
- Full stats page with multiple integrations
- GIF showcase
- Perfect responsive design across all breakpoints

**Future Versions:**
- Blog functionality
- Additional project types/sections
- More sophisticated stats/visualizations
- Search functionality

### Don't Get Stuck On:
- Perfect design from the start
- Every integration working perfectly
- Comprehensive project catalog
- Advanced features

Remember: Ship something real, iterate from there.
